## 3. 记录格式

### 学习记录 (LEARNINGS.md)

```markdown
## [LRN-20260228-001] 简短标题（15字内）

**优先级**：high | **状态**：pending | **标签**：#skills #安装

### 问题
一句话描述问题是什么

### 解决
一句话描述解决方案

### 详情（可选）
- 具体场景：...
- 原因分析：...
- 相关文件：...

---
```

### 错误记录 (ERRORS.md)

```markdown
## [ERR-20260228-001] 错误类型

**优先级**：high | **状态**：pending | **标签**：#API #网络

### 错误信息
```
实际的错误输出（简要）
```

### 原因
一句话描述原因

### 解决方案
一句话描述如何解决

### 复现步骤（可选）
1. 执行命令 xxx
2. 出现错误 xxx

---
```

### 功能请求 (FEATURE_REQUESTS.md)

```markdown
## [FEAT-20260228-001] 功能名称

**优先级**：medium | **状态**：pending | **标签**：#增强

### 需求
用户想要什么功能

### 场景
用户在什么情况下需要

### 建议实现（可选）
- 方案1：...
- 方案2：...

---
```

## 4. 共享学习机制

### 什么值得共享？

| 共享 | 不共享 |
|------|--------|
| skill 安装经验 | 用户隐私信息 |
| 网络问题解决方案 | 单个用户的偏好 |
| 通用命令技巧 | 领域专业知识 |
| 配置文件最佳实践 | 临时性错误 |

### 共享流程

```
bot 发现通用经验
        ↓
写入 /home/NanoBot/shared/learnings/SHARED.md
        ↓
其他 bot 启动时自动加载共享内容
        ↓
所有 bot 共同进步
```

### 共享记录格式

```markdown
## [SHARED-20260228-001] 通用经验标题

**适用**：所有 bot | **标签**：#通用 #安装

### 问题
通用问题描述

### 解决
通用解决方案

### 来源
bot-03 于 2026-02-28 发现

---
```

## 5. 联动其他 Skill

### 与 policy-reader 联动

```
用户发政策 → policy-reader 解读
        ↓
解读时记录预测到 POLICY_TRACKS.md
"预测：3个月内XX行业会有补贴"
        ↓
3个月后 bot 主动验证
"上次预测准吗？"
        ↓
用户反馈 → 记录到 LEARNINGS.md
"政策预测准确度：80%"
```

### 与 find-skills 联动

```
skill 安装失败 → 记录到 ERRORS.md
        ↓
自动参考 find-skills 的备选方案
"网络超时用 wget 手动下载"
        ↓
下次安装时自动应用
```

### 与用户画像联动

```
用户纠正建议 → 记录到 LEARNINGS.md
"用户风险偏好=保守，不推荐高风险"
        ↓
同步更新 MEMORY.md 用户画像
        ↓
下次建议自动应用新画像
```

## 6. 触发场景速查

| 场景 | 动作 | 写入文件 |
|------|------|----------|
| 命令执行失败 | 记录错误 + 解决方案 | ERRORS.md |
| 用户说"不对"、"其实是" | 记录纠正内容 | LEARNINGS.md |
| 用户想要缺失功能 | 记录需求 | FEATURE_REQUESTS.md |
| API 调用失败 | 记录错误 + 备选方案 | ERRORS.md |
| 发现知识过时 | 记录更新内容 | LEARNINGS.md |
| 找到更好方法 | 记录最佳实践 | LEARNINGS.md |
| 通用经验 | 共享给所有 bot | SHARED.md |

---

## 7. 状态流转

```
pending（待处理）
    ↓
in_progress（处理中）
    ↓
resolved（已解决）或 promoted（已提升到文档）
```

---

## 9. 使用建议

1. **立即记录**：发现问题后立即记录，不要等
2. **定期复习**：每周回顾 pending 状态的记录
3. **及时更新**：问题解决后更新状态为 resolved
4. **主动共享**：发现通用经验写入 SHARED.md
5. **联动应用**：记录后思考能否联动其他 skill
