# 共享见解系统

将有价值的讨论整理成文档，存放在共享目录，供所有bot参考。

## 触发场景

| 功能 | 触发词示例 |
|------|-----------|
| 创建 | "记录一下这个话题"、"把这次讨论整理到共享见解" |
| 读取 | "看看共享见解中的XXX" |
| 更新 | "更新共享见解中的XXX"、"在XXX里追加新内容" |
| 删除 | "删除共享见解中的XXX" |
| 列表 | "共享见解有哪些"、"列出共享见解" |
